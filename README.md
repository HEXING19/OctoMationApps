# OctoMationApps

OctoMationApps 是 [OctoMation](https://github.com/flagify-com/OctoMation) （雾帜智能自动化产品）的社区化产品的应用集合。

## OctoMation 介绍

OctoMation 是一个 SOAR(Security Orchestration Automation Response)系统。它提供了一个集成化的工作台,可以建立自动化的工作流程以响应安全事件。

OctoMation 的特点包括:

- 简单且直观的图形化界面以建立自动化流程
- 支持各种安全产品和服务的连接集成
- 强大的自定义功能,支持 Python 脚本进行扩展
- 社区驱动,用户可以共享工作流程和app

## OctoMationApps 

OctoMationApps 提供了一系列为 OctoMation 建立的自动化应用程序和脚本。这些应用程序可以直接导入 OctoMation 中,实现各种自动化任务。
